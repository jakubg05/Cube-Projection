# Cube Projection

Basic projection of a cube

<b>Move:</b> <i>w, a, s, d, shift, spacebar</i> <br/>
<b>Rotate:</b> <i>(Up, Down, Left, Right) arrows</i> <br/>
<b>Increase/Decrease focal length:</b> <i>i, k</i> <br/>
<b>Increase/Decrease FOV:</b> <i>u, j</i><br/>
<br/><br/>
